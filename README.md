# QR Data Bridge Sender

二维码幻灯片发送器 - 将任意文件转换成二维码幻灯片进行传输

## 📖 简介

这个工具可以将任何二进制文件切分成数据块，每个数据块生成一个二维码，然后以幻灯片的形式播放。配合 **QR Data Bridge** 安卓应用使用，实现通过摄像头进行离线大数据传输。

## ✨ 功能特性

- 📁 **支持任意文件** - 文本、图片、文档、视频等
- 🔧 **可调数据块大小** - 100-2000 字节，控制每次传输的数据量
- ⏱️ **可调播放速度** - 0.1-10 秒/帧，适应不同扫描速度
- ⏮️ **手动控制** - 上一个/下一个/跳转/重放
- ▶️ **自动播放** - 幻灯片模式连续播放
- 📊 **进度显示** - 实时显示传输进度
- 🔍 **数据预览** - 查看当前数据块内容

## 🚀 快速开始

### 系统要求

- Python 3.10 或更高版本
- tkinter (通常随 Python 安装)
- Windows / macOS / Linux

### 安装依赖

```bash
# 创建虚拟环境（推荐）
python3 -m venv venv
source venv/bin/activate  # Linux/macOS
# 或 venv\Scripts\activate  # Windows

# 安装依赖
pip install -r requirements.txt
```

### 运行程序

```bash
python qr_sender.py
```

## 📱 使用方法

### 1. 选择文件

点击 "📁 选择文件" 按钮，选择要传输的文件。

### 2. 设置数据块大小

- **小文件** (＜1MB): 200-300 字节
- **中等文件** (1-10MB): 300-500 字节
- **大文件** (＞10MB): 500-1000 字节

**注意**: 
- 数据块越小，二维码越简单，扫描越快，但总二维码数量越多
- 数据块越大，二维码越复杂，扫描越慢，但总二维码数量越少

### 3. 设置播放速度

根据安卓应用的扫描速度调整：
- **快速扫描**: 0.5-1.0 秒/帧
- **正常速度**: 1.0-2.0 秒/帧
- **慢速扫描**: 2.0-5.0 秒/帧

### 4. 开始传输

1. 点击 "▶️ 播放" 开始自动播放
2. 使用安卓应用扫描屏幕上的二维码
3. 如需暂停，点击 "⏸️ 暂停"
4. 如需回退，点击 "⏮️ 上一个"
5. 如需重放，点击 "🔄 重放"

## 🎯 使用场景

### 场景 1: 离线文件传输

在没有网络的环境中，通过二维码将文件从电脑传输到手机。

### 场景 2: 安全数据传输

通过可见光传输，不经过网络，避免数据被截获。

### 场景 3: 跨设备数据同步

将配置、密钥等数据从一台设备同步到另一台设备。

## 📊 传输效率参考

| 文件大小 | 数据块大小 | 二维码数量 | 传输时间 (2 秒/帧) |
|---------|-----------|-----------|------------------|
| 100 KB  | 200 字节   | 512       | ~17 分钟         |
| 500 KB  | 300 字节   | 1,707     | ~57 分钟         |
| 1 MB    | 500 字节   | 2,098     | ~70 分钟         |
| 10 MB   | 1000 字节  | 10,486    | ~5.8 小时        |

**建议**: 适合传输小到中等大小的文件（＜5MB）

## 🔧 高级功能

### 元数据

第一个二维码包含文件元数据：
- 文件名
- 总块数
- 数据块大小
- 协议版本

### 数据编码

- 使用 Base64 编码二进制数据
- 高纠错级别 (H) - 可恢复 30% 数据损坏
- QR 版本 10 - 最大容量约 1700 字节

### 手动控制

- **上一个**: 回退到上一个二维码
- **下一个**: 跳到下一个二维码
- **跳转**: 直接跳转到指定位置
- **重放**: 从头开始播放

## ⚠️ 注意事项

1. **屏幕亮度**: 确保屏幕亮度足够，避免反光
2. **扫描距离**: 保持适当的扫描距离（15-30cm）
3. **稳定性**: 播放过程中不要移动设备
4. **光线**: 确保环境光线充足
5. **电量**: 传输大文件时确保设备电量充足

## 🐛 故障排除

### 二维码无法扫描

- 降低数据块大小
- 增加播放速度
- 调整屏幕亮度
- 检查扫描距离

### 传输完成后文件损坏

- 检查是否有二维码被跳过
- 使用手动模式重新扫描损坏部分
- 确保扫描环境稳定

### 程序无法启动

- 检查 Python 版本（需要 3.10+）
- 安装依赖：`pip install -r requirements.txt`
- 检查 tkinter 是否安装
